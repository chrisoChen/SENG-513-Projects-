Name: Chris Chen

Description: A simple calculator web-app built to 
experiment and strengthen JavaScript knowledge 

Tested At The Following Browsers:
 - Google Chrome Desktop
 - Google Chrome Mobile
 - Firefox Desktop 

Note:

- Because eval() is used to determine calculations, this app
is not recommended to deploy because of security risks and 
the unpredictability of this web-app. See more at the following: 
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/eval#Never_use_eval!  

- Green Color for Scroll bar on calculator screen does 
not appear on Firefox browsers at time of testing, appears on Google Chrome 
browsers.
