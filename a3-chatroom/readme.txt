Name: Chris Chen

Description: A simple chat app built to grasp and experiment with 
Node.js, express, and socket.io

Instructions to run: 
    - ensure express and socket.io are installed: run npm install to install required dependencies
    - cd into public directory
    - run "node index.js" on terminal to start server server: will start connection on localhost:3000

	- Tested with 2 concurrent users using Google Chrome and Firefox
